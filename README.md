# graphql-test

Este repositorio contiene un proyecto hecho con Java 17 y SpringBoot, implementando GraphQL.
Este proyecto tiene tanto ejemplos de Query como de Mutations.

## Tecnologias usadas
- **MySQL**
- **Java 17**
- **Spring Boot**
